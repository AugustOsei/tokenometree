How it works

Paste your API key

One key per provider. No browser extensions, no OAuth dance, no forwarding proxies. Tokenometree reads usage directly from each provider's API.

Total spend across all providers. Per-model, per-day, per-project breakdowns. One number that answers "how much am I spending on AI this month?"
